Change Default Branch on GitHub (Website): Go to your repository on GitHub. Click on the "Settings" tab (⚙️ top right of the repo). In the left sidebar, click “Branches”. Under Default branch, click the "Edit" button. Select the new default branch from the dropdown list. Click "Update", then confirm by clicking "I understand, update the default branch" .
